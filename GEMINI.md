# Project Overview

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app). It appears to be a web application for "SMBShield", which provides AI-powered security intelligence for European SMBs.

The application is built with [React](https://reactjs.org/) and [TypeScript](https://www.typescriptlang.org/). It uses [Tailwind CSS](https://tailwindcss.com/) for styling and includes a variety of UI components from `@radix-ui` and other libraries. [Framer Motion](https://www.framer.com/motion/) is used for animations.

The project structure is well-organized, with a clear separation of pages, components, and other files.

# Building and Running

To get the development server running, use the following command:

```bash
npm run dev
```

To create a production build, use:

```bash
npm run build
```

And to start the production server, use:

```bash
npm run start
```

# Development Conventions

The codebase is written in TypeScript and uses modern React features like hooks. Components are organized by feature and are located in the `components` directory. The project uses a custom theme provider for light/dark mode.clear

The code is well-formatted and uses a consistent style. It is recommended to follow the existing coding style and conventions when contributing to the project.
